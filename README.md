# Sakli_Rota
Full-stack map-based web app for sharing and discovering locations
Saklı Rota, kullanıcıların harita üzerinden konum ekleyerek keşfedilmemiş veya favori yerlerini paylaşabildiği bir full-stack web uygulamasıdır. Proje geliştirme sürecinde frontend ve backend birlikte ele alınmış, harita servisleri, kullanıcı yetkilendirme ve CRUD işlemleri üzerine pratik kazanılması hedeflenmiştir.
